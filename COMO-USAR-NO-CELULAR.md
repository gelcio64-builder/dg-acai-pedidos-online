# 📱 Como usar o DG Açaí no seu Celular

## 🚀 Opções para acessar o site no celular:

### 1️⃣ **Opção Mais Fácil - Netlify (Recomendado)**
1. Acesse: https://www.netlify.com/
2. Clique em "Sign up" (gratuito)
3. Faça login com GitHub ou Google
4. Arraste o arquivo `dg-acai-mobile.html` para a área de upload
5. Aguarde alguns segundos
6. Copie o link gerado (exemplo: https://dg-acai-123.netlify.app)
7. Compartilhe o link com seus clientes!

### 2️⃣ **Opção Rápida - GitHub Pages**
1. Crie uma conta no GitHub: https://github.com
2. Crie um novo repositório chamado "dg-acai"
3. Faça upload do arquivo `dg-acai-mobile.html`
4. Renomeie para `index.html`
5. Vá em Settings > Pages
6. Ative GitHub Pages
7. Use o link gerado

### 3️⃣ **Opção Simples - Vercel**
1. Acesse: https://vercel.com/
2. Faça login com GitHub
3. Clique em "New Project"
4. Faça upload do arquivo
5. Use o link gerado

## 📋 **Passo a Passo Detalhado (Netlify)**

### Passo 1: Preparar o arquivo
- O arquivo `dg-acai-mobile.html` já está pronto
- Renomeie para `index.html` (opcional, mas recomendado)

### Passo 2: Fazer upload
1. Abra https://www.netlify.com/
2. Clique em "Sign up" (gratuito)
3. Escolha "Sign up with GitHub" ou "Sign up with Google"
4. Faça login
5. Na tela inicial, arraste o arquivo `dg-acai-mobile.html` para a área "Drag and drop your site output folder here"
6. Aguarde o upload

### Passo 3: Obter o link
1. Após o upload, você verá um link como: `https://amazing-name-123456.netlify.app`
2. Clique no link para testar
3. O site deve abrir perfeitamente no celular!

### Passo 4: Personalizar (opcional)
1. Clique em "Site settings"
2. Vá em "Change site name"
3. Escolha um nome como: `dg-acai-pedidos`
4. Seu link ficará: `https://dg-acai-pedidos.netlify.app`

## 📱 **Como testar no celular:**

1. **Envie o link por WhatsApp** para você mesmo
2. **Clique no link** no seu celular
3. **Teste todas as funcionalidades:**
   - Adicionar pedido
   - Escolher produto
   - Marcar complementos
   - Gerar pedido

## 🎯 **Dicas importantes:**

- ✅ O site funciona offline após carregar
- ✅ Funciona em qualquer celular (Android/iOS)
- ✅ Não precisa de app, só o navegador
- ✅ Link pode ser compartilhado facilmente
- ✅ Totalmente gratuito

## 🔧 **Se algo não funcionar:**

1. **Limpe o cache** do navegador
2. **Tente outro navegador** (Chrome, Safari, Firefox)
3. **Verifique a conexão** com internet
4. **Recarregue a página** (puxe para baixo)
